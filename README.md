# 정보처리산업기사 과정평가 공개문제(실기) 실습과정(1)

## 전체적인 설명
#### header, nav, section, footer 부분을 4개의 JSP파일로 만들어 제작.
#### 메인 JSP파일에 링크로 연결하여 하나의 페이지 제작.
#### CSS를 이용하여 전체적으로 margin, padding 0으로 지정.
#### ul, li 태그는 list-style을 none으로 지정. a 태그는 텍스트 색상을 #ffffff, text-decoration을 none으로 지정.

# nav
## 코드 설명
#### ul, li 태그를 이용하여 목록을 만들어줌.
#### a태그의 href속성을 이용하여 연결할 주소를 지정해줌.(다른 페이지를 제작 안되었기떄문에 "#"으로 대체)
#### float을 left로 지정. 크기, 배경색, 텍스트 조정해줌.

# section
## 코드 설명
#### 전체적인 크기와, 배경색 지정.
#### position을 fixed로 지정하여 고정시켜줌.
#### 제목 부분을 padding을 상,하에 20px, text-align을 center로 지정.
#### 문단부분을 padding을 전체적으로 20px 지정.
